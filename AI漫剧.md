## 漫剧概念

### 基础认知

漫画短剧，它和漫画、动画有最核心的区别，受众人群不同。它是面向疲劳人群的、高
情绪密度的娱乐快餐，它和短剧一样提供**紧凑情节**，让观众快速获得爽快感，用户不会
太在意画面清晰度、人物或部分画面会不会崩，对细节追求较低

### 漫剧品类

- #### AI仿真人剧

  基本利用AIGC技术制作人物场景超写实，效果逼真角色有吩咐的动态交互场景，有多音色对话

- #### 表情包动态漫（沙雕漫）

  以熊猫头等贴图作为主要素材制作，角色有简单的动态交互场景，有多音色对话

- #### 静态漫画

  静态图片配以有声书或单一角色旁白解说，角色无动态交互场景，无音色对话

- #### 2d/3D动画慢剧

  2D/3D平面图形或3D图像制作的动画角色有简单的动态交互场景，有多音色对话

### 动态分镜

- 动态分镜：可验证的最小交付单元，包含了90%的创作价值（含运镜，节奏，情绪，信息释放）

- “影片的动态草稿”，它不想最终的成片那样精致，但能精准的传递创意核心

- 最终成片：动态分镜的加工（包含了后期，配音特效，渲染，字幕）

### 漫剧和AI短片核心区别

| 短片               | 漫剧                 |
| ------------------ | -------------------- |
| 一次性叙事         | 系列化叙事           |
| 图像生成技术       | 前期资产设定         |
| 单次视觉效果       | 可重复利用长期体系化 |
| 用完即弃，只用一次 | 几十集到100集        |



## 漫剧市场

### 人才需求：

#### 导演:

1. 具备出色的写故事能力，能独立构建故事大纲，分集剧情及人物小传，剧本节奏紧凑有短剧，动漫，或网文。
2. 具备扎实的导演基本功，熟悉短剧制作全流程，熟悉AI工具的视觉生成逻辑，掌握镜头语言，分镜设计，剪辑节奏等专业知识。
3. 具备出色的流程管理，标准化建设和项目管理能力，能够清晰规划项目的制作需求，为团队制作有效的协作规范，进行有效的进度跟踪管理。
4. 。熟悉AIGC创作工具（如 Runway，PIKA，KLING，ComfyUI，MiJJourney,Sora,AnimateDiff等），具备AI镜头或角色生成经验，有AIGC动画短片。

#### 编剧

1. 负责AI漫剧/短剧仙姑的原创剧本创作或改编，编写分集大纲，人物设定和分镜台词。
2. 网感好，熟悉短句/爽剧节奏，能根据爆款题材（如系统类，逆袭类，都市类，重生类）高效产出剧情。
3. 能与制作团队协作，根据AI制作特点优化剧本结构。
4. 拥有较强的爽点设计能力（打脸，逆袭，升级，反转），保证剧情节奏快，冲突强。
5. 剧本审编：负责剧本的审核工作，跟进编剧创作过程，从题材，大纲，剧本创作等维度提出意见和建议，对剧本完成质量负责。

#### 分镜师

1. 具备良好 的美术功底和手绘能力，对构图、透视、光影有深刻的理解，能快速清晰地表达画面。
2. 镜头感强，熟悉影视语言和动画叙事手法，能通过镜头有效的传达情节和情绪。
3. 熟练使用Photoshop，对市面上主流的AIGC类工具有所了解。
4. 热爱动漫事业，漫画行业，对漫剧、短剧等视频平台上的内容有浓厚的兴趣。

#### AI绘画师

1. 具备优秀的审美能力，对构图，色彩，光影有深刻理解。
2. 熟练使用市面上主流AIGC类工具，包括即梦/可灵/海螺/Sora/Stable/Diffusion/Midjourney/ComfyUI等，能够通过精准的Prompt（提示词）控制画风和细节。
3. 具备良好的美术基础和绘画能力，可以对AI生成的图片进行细化修改和二次创作。
4. 了解漫剧的叙事节奏，镜头语言和基本镜头运动原理。
5. 是ACGN(动画，漫画，游戏，小说)文化爱好者，网感好，能精准把握网络热点和用户喜好。

#### 剪辑师

1. 精通Premiere，After Effects及剪映等主流剪辑软件，功底扎实。
2. AIGC工具熟练：深度使用过至少一种主流AIGC工具，如即梦，Stable Diffusion,Midjourney等，并理解Prompt工程的基本逻辑。
3. 卓越的审美：对画面构图、色彩、节奏和音乐有敏感的感知力，能独立把控影片风格。
4. 高效协作与沟通：能快速理解创意意图，适应快节奏，多迭代的项目环境。

## 总结

### 题材选择：

新人不推荐修仙题材，科幻题材，需要制作大量特效画面，费时费力

### 规避敏感内容

如暴力，血腥，色情及敏感的对白

### 运镜选择

减少使用复杂运镜，前期尽量以固定镜头
