# MJ参数详解

| 参数 | 实例 | 作用建议 |
| --- | --- | --- |
| --v+[空格]+(1-7) | --V7 | 选择模型版本。--v适合在真实感，提示词理解、艺术出创作的模型； |
| --niji+[空格]+(4-6) | -- niji 6 | --niji：为动漫和插画风格的优化模型 |
| --ar+[空格]x:y | --ar 1:1(方形)/ --ar 16:9(横构图)/--ar 2:3(竖构图) | 控制画面长宽，是构图的关键参数 |
| --q+[空格]<0.25-1(默认)> | --q .25(低质快速)/ --q 1(默认)/ --q 2(高质量慢速) | 控制渲染时间与细节：质量越高画面越清晰相对时间也越长，值越小适合抽象的图像，值越大适合细节性画面 |
| --chaos+[空格]<0-100> | --c 0(稳定)/ -c 80(多变) | 控制随机性，高值更具创意但可控性降低 |
| --seed+[空格]<数值> | --seed 12345 | 固定随机种子，保证重复生成一致的结果，适合系列或版本对比 |
| --stop+[空格]<10-100> | --stop <75>;设置75获得草图效果，100生成完成细节 | 体检终止生成，得到更“松散“”手绘“或低完成度图像 |

## 进阶参数

| 参数 | 实例 | 作用建议 |
| --- | --- | --- |
| --weird+[空格]<0-3000> | --wired <1000> | 生成”超现实/奇怪“效果，越高越离谱，创意设计用200-500，写实场景建议<=50 |
| --tile | --tile | 平铺画面(生成无缝重复团)用户壁纸、面料设计，配合简单元素效果更加 |
| --no | --no people | 排除指定元素，配合主体描述使用，避免过渡过滤导致画面残缺 |
| --stylize+[空格]<0-1000> | --s 500/ --stylize 500 | 产品摄影用50-200，艺术创作用800-1000；0-200适合商业设计，高于600后会出现符合运动轨迹的模糊效果，高值 800-1000适合抽象艺术 |
| --style | --style raw：渐少70%的默认美化，保留原始纹理（适合产品摄影、建筑可视化）；-exp[空格]<10-100>：强化笔触流动感（适合插画、概念艺术） | 调整模型美学倾向，写实场景用raw模式，动漫风格用niji模式 |

### 生成接近真实照片，介绍AI艺术感

推荐后缀：

```markdown
--ar 3:4 --stop (70-80) --s 100 --v 6.1 
--raw --ar 3:4 --v 7
```

```markdown
a portrait photo of an urban woman, shot with studio lighting,8k,--raw --ar 3:4 --v 7
```

生成动漫风格手机壁纸

推荐后缀：

```markdown
版本风格：
-- niji 6 --s 200 --ar 9:16
-- v7 --s 800 --ar 4:3
```

```markdown
a girl sitting by window,raining,otomo katsuhiro style, --ar 9:16 --s 200 
a girl sitting by window,raining,otomo katsuhiro style, --ar 9:16 --s 800 --v 7
```

生成可无缝拼接的图案，并对图案进行延展

```markdown
--tile --ar 1:1 --c 5 --q 2
```

```markdown
a bird on the branches --ar 1:1 --c 5 --q 2 --tile
```
