



## 商业落地四步法

### 需求洞察

找准用户的核心痛点，明确真实需求方向

```
我想做一个（个人母婴手册产品）相关的知识产品，请你帮我分析：
1. 哪些用户存在真实的需求？
2. 他们目前的困难是什么？
3. 为什么他们愿意为解决方案付费？
```



### 方案设计

搭建落地框架，突出差异化与可执行性

```
基于这个需求，请你帮我设计一个知识型产品（例如小课程，模板包）。内容包含：
1.产品定位
2.用户能获得的结果
3.课程大纲（3-5节）
4.MVP如何去做
```



### 执行落地

拆解关键动作低成本快速推进落地

1周：需求访谈

1周：内容制作

1周：发布与试卖

1周：复盘优化

```
请为我设计一个从0-1的行动计划（适合5人执行）包含：
1.1周：需求访谈
2.1周：内容制作
3.1周：发布与试卖
4.1周：复盘优化
```



### 验证与优化

用数据验证效果，针对性调整

- ```
  请基于我的产品，帮我设计3个最简单的市场验证方式
  
  并告诉我
  
  - 如果验证结果是正向的，下一步做什么
  - 如果验证结果一般，我要如何调整
  ```

  

模板

1. ```
   请你作为活动策划顾问，帮我做需求洞察，主题是（城市轻徒步计划），请输出 
   
   1. 目标用户是谁？
   2. 他们的痛点是什么？
   3. 当前市场上已有的解决方案是什么、
   4. 可悲优化的机会点？
   ```





```
   
   
   基于上面的需求洞察，帮我设计一个活动方案，要求：
   
   1. 提供3个可行方案
   
   2. 每个方案包含：
   
      1. 价值主张
   
      2. 核心功能
   
      3. 差异化
   
      4. MVP
   
         
```
